Projects and Contributions

Projects

DSIP
View code

DSIP is my thesis project (final engineering project). It's an app for creating distributed computing service, that is easily extensible. It utilizes gRPC framework, GoCV (OpenCV wrapper) and connects to PostgreSQL Database.

Squat
View code

Squat is an application that provides simple SQL data generation functionality. It generates synthetic SQL data based on the table definition, that is gathered from the DBMS. Squat supports IBM Informix and PostgreSQL.
